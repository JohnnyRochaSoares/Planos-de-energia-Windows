# **Planos de Energia do Windows 11**



* Este projeto é designado aos utilizadores, especialmente *"gamers"* em computadores portáteis, para poderem usar o portátil no modo Desempenho (sem focar na bateria), modo Poupança de Bateria estrema ou modo Médio (mais focado em poupança de bateria).







### Funcionalidades:





##### Modo Poupança de Bateria:



* Mínima Utilização do Processador: 0%;
1. Máxima Utilização de Processador: 5%;
* Modo Turbo: Desligado;
* Suspensão Rápida: 1 minuto;
* Ecrã Desligado: 1 minuto;
* Economia de Bateria do Windows: Ligoado;
* Plano de energia da Placa Gráfica: Económico.









##### Modo Médio:



* Mínima Utilização do Processador: 0%;
* Máxima Utilização de Processador: 75%;
* Modo Turbo: Desligado;
* Suspensão Rápida: 1 minuto;
* Ecrã Desligado: 1 minuto;
* Economia de Bateria do Windows: Ligado;
* Plano de energia da Placa Gráfica: Económico.









##### Modo Desempenho:



* Mínima Utilização do Processador: 100%;
* Máxima Utilização de Processador: 100%;
* Modo Turbo: Ligado;
* Suspensão Rápida: 0 minutos;
* Ecrã Desligado: 0 minutos;
* Economia de Bateria do Windows: Desligado;
* Plano de energia da Placa Gráfica: Desempenho máximo.

# **Instalação:**



### Pelo GitHub.com:



1. Vá ao link "https://github.com/JohnnyRochaSoares/Planos-de-energia-Windows-11/releases/tags";
2. Vá à versão mais recente;
3. Descarregue;
4. Extraia o ficheiro;
5. Dê um duplo clique para ativar o modo que deseja.







### Pelo Git:



NOTA: Requer Git instalado no Computador.



1. Abra o terminal como administrador;
2. Execute o comando "git clone https://github.com/JohnnyRochaSoares/Planos-de-energia-Windows-11.git";
3. Execute o comando "cd "Planos-de-energia-Windows-11"".
